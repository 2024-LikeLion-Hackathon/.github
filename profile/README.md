# Emolog (Emotion + log)
사람들이 ‘**자기감정**’에 솔직해질 수 있는 메모장

인공지능을 통해

누가 내 얘기를 듣는다는 **부담감 없이**, 하지만 들어주는 사람이 있다는 **위로감**을 얻고,

이야기를 분석해서 되묻는 질문들을 통해 

**자기 자신을 더 들여다보고 이해**할 수 있도록 한다.
